# Plano de Vida

Projeto local para acompanhar diariamente um plano de vida cristão inspirado na rotina de um supranumerário da Opus Dei.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Estrutura

- `index.html`: página principal.
- `src/styles.css`: estilos visuais.
- `src/data.js`: lista de práticas e categorias.
- `src/app.js`: lógica dos checkboxes, data corrente, progresso e notas.
